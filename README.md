Data Cleaning

Cohort Analysis

Retention Trend Analysis

🔍 Key Insights

Month-to-month contracts have highest churn.

Long-term contracts show better retention.

Customers with higher monthly charges churn more.

New customers leave more often in the first few months.

💡 Recommendations

Encourage long-term contracts

Offer loyalty rewards

Provide discounts for high charges

Improve customer onboarding

Create targeted retention campaigns
